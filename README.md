## Installation Guide 

- have **node.js v16.6 or higher**

- in `./config.json` Adjust the BOT TOKEN and PREFIX

- `npm install` to Install the depedences ( or `npm install discord.js enmap moment` )

- `node .` to start the Bot ( or `node index.js` / `npm start` )
